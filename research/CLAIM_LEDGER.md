# Claim Ledger

## Status Rules

This ledger records claims **as the manuscript currently states them**. It does
not silently repair grammar, narrow scope, or confer validity. “Unestablished”
means the repository currently contains no adequate proof, citation, model, or
experiment for the stated scope. Ordinary demo execution can establish only an
implementation observation, not a cryptographic security theorem.

The RPS, Hash Functions, Trust Establishment, and Tic-Tac-Toe slices now have
small verified bibliographies with claim-scoped support and non-support
boundaries. The Tic-Tac-Toe public checker, fixed-DAG evaluator, and Boolean
Core also have recorded implementation evidence. Every other chapter retains its Phase 0
citation status unless an entry explicitly says otherwise. A related citation
does not prove a different construction.

`AC-TRUST-01` through `AC-TRUST-14` preserve former-draft claims. Their old
line references intentionally record the pre-repair source; the active chapter
does not silently restate them.

## Immediate Ledger Conclusions

1. The RPS chapter now contains one narrow conditional correctness proposition
   with a short complete argument. It still contains no formal cryptographic
   security game or reduction; hiding and binding remain explicitly limited
   security arguments.
2. RPS is now the clearest end-to-end protocol specification and has a sparse
   cited literature chain. Its actual implementation evidence must come from
   the final build/test report. Poker remains the strongest pre-existing
   executable construction, limited to an honest centralized animation.

   key-disclosure, public-beacon, and raw-mutual-information claims from active
   prose while preserving them above for traceability. Its active result is a
   taxonomy and set of conditional boundaries, not a key protocol.
4. The most consequential remaining unsupported claims are the Introduction's
   blanket information-theoretic/quantum guarantees and the poker text's
   distributed privacy/verifiability language.
5. Phase 0 changed no manuscript claim. Later bounded slices preserve original
   wording, introduce only narrower model-specific claims, and record the
   evidence boundary for each.
